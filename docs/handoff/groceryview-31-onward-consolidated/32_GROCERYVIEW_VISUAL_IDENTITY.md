# 32 — GroceryView Visual Identity

## Brand concept

```text
GroceryView: verified price intelligence for everyday shopping.
```

## Visual metaphor

```text
A clean Nordic market dashboard.
```

The website should make grocery prices, store locations, deals, pharmacy OTC prices, and fuel prices understandable.

## Brand keywords

```text
fresh
verified
local
smart
calm
fair
transparent
source-backed
money-saving
```

## Main visual direction

```text
Background:
warm oat / off-white, not pure grey

Primary color:
deep forest green

Accent:
fresh lime / mint for positive saving

Warning:
amber for medium confidence or caution

Danger:
rose/red for price increase or low confidence

Info:
blue/indigo for pharmacy safety and source explanation

Fuel:
amber/slate/blue depending on grade/operator
```

## Typography direction

Use one clean sans-serif family for all UI.

Suggested stack:

```css
font-family: Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```

Recommended weights:

```text
400 body
500 secondary emphasis
600 labels
700 buttons/cards
800 page titles
900 hero/KPI numbers only
```

## Type scale

```text
Hero: 56–72px desktop, 38–44px mobile
Page H1: 40–52px desktop, 32–38px mobile
Section H2: 24–32px
Card H3: 18–22px
Body: 16px
Small: 14px
Micro/evidence: 12px
```

## Visual surface style

Use:

```text
soft rounded cards
subtle borders
calm shadows
large whitespace
clear grouping
one strong CTA per section
```

Avoid:

```text
too many bright backgrounds
all cards same weight
too many saturated colors
huge borders
technical labels as headings
unnecessary gradients everywhere
```

## Logo direction

Simple MVP logo:

```text
GV mark in dark forest green
small price/index line motif
wordmark: GroceryView
tagline: Verified price intelligence
```

Possible icon ideas:

```text
leaf + chart line
shopping basket + signal line
map pin + price tag
```
