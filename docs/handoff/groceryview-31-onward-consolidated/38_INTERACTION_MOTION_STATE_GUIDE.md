# 38 — Interaction, Motion, and State Guide

## Interaction states

Every interactive element needs:

```text
default
hover
active/pressed
focus-visible
selected
disabled
loading
error
```

## Motion

Use subtle motion only:

```text
card hover lift: 2–4px
button press: slight scale
drawer open: 150–220ms
skeleton shimmer: subtle
chart tooltip fade: 100ms
```

Avoid:

```text
flashy animations
attention-grabbing ad motion
bouncy charts
overusing gradients
```

## Loading states

Use skeletons for:

```text
search results
charts
product cards
map panels
watchlist
```

## Selected state

When a filter is selected:

```text
clear chip
clear remove button
visible page summary
```

Example:

```text
Viewing: Grocery · Stockholm · Meat · Willys · 1Y
```

## Error states

Use:

```text
plain message
specific problem
next action
```

## Empty states

Use:

```text
friendly icon
title
short body
3 next actions
```
