# 34 — Page Redesign Blueprints

## Home redesign

Goal:

```text
Make GroceryView attractive in the first 5 seconds.
```

Layout:

```text
Hero
├── Large headline
├── Simple subtitle
├── Big search
├── 3 CTAs
└── Trust chips

Domain cards
├── Grocery
├── Pharmacy OTC
└── Fuel

Today panel
├── Real deals
├── Price movers
└── Map/store preview

Market snapshot
How it works
Ad slot
Footer
```

Hero copy:

```text

Find real deals, compare stores, and see the evidence behind every price.
```

Trust chips:

```text
Verified sources
Freshness labels
No fake prices
Source confidence
```
