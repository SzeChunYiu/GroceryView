# 35 — Component Polish Specs

## PageQuestionHeader

Should include:

```text
eyebrow
question
h1
subtitle
actions
evidence
```

Visual style:

```text
large white card
warm background
one strong CTA
evidence chips below subtitle
```

## ProductCard

Must include:

```text
image
domain badge
name
brand
package
price
unit price
deal label
store/chain
freshness/confidence
Open product
```

Visual style:

```text
white card
image on left/top
price in forest green
deal label as pill
evidence in muted row
hover lift
```

## DealCard

Must include:

```text
Real Deal / Fair Discount / Not Really a Deal
product name
current price
historic comparison
nearby comparison
reason
evidence
Open product
```

## KpiCard

Must include:

```text
label
value
delta
plain-language meaning
click action
```

## EvidenceStrip

Must include:

```text
source
last observed
freshness
confidence
observation count
```

Do not make it visually louder than the main content.

## ChartShell

Must include:

```text
question
title
summary
chart
legend
actions
evidence
table fallback
```

## AdSlot

Must include:

```text
label: Advertisement
reserved height
neutral style
clear separation
```

## EmptyState

Must include:

```text
what is missing
why it is missing
what to do next
```
