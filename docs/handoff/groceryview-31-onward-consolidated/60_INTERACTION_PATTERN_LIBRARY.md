# 60 — Interaction Pattern Library

## Product preview card

Trigger:

```text
Click "Quick view" or tap product card preview area.
```

Content:

```text
Product image
Name
Brand/package
Best known price
Deal label
Where to buy top 3
Mini price history
Freshness/confidence
Actions:
- Open product
- Add to watchlist
- Compare stores
```

## Store preview card

Content:

```text
Store name
Chain
Address
Opening hours
Coverage
Top deals
Actions:
- Open store
- Search products here
- Open map
```

## Category preview card

Content:

```text
Category
Weekly / 3M / 1Y
Cheapest chain
Best deal
Confidence
Actions:
- Market view
- Browse products
- Search deals
```

## Deal preview card

Content:

```text
Deal label
Product
Current price
Compared with usual price
Compared with nearby alternatives
Why this label
Actions:
- Open product
- View similar
- Add to watchlist
```

## Fuel station preview

Content:

```text
Station name
Chain
Distance
Grade availability
Operator price caveat
Actions:
- Open station
- Open map
- Compare grade
```

## Pharmacy OTC preview

Content:

```text
Product
EAN
Lowest public catalog row
Compared chains
Safety boundary
Actions:
- Open OTC page
- Set alert
- View source
```

## Evidence drawer

Use for any price/deal/index:

```text
Source
Last observed
Rows
Freshness
Confidence
Known limitations
Link to data sources
Optional backstage link for admin
```

## Save-view drawer

For market/map dashboards:

```text
View name
Domain
Filters
Alert option
Save
```

## Report issue modal

Use modal only because it is a focused task.

Fields:

```text
What looks wrong?
Product/store/source
Optional screenshot/source
Submit
```
