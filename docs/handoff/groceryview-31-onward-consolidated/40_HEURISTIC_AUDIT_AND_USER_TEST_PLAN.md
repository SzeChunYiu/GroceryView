# 40 — Heuristic Audit and User Test Plan

## Heuristic audit

Review every page against:

```text
Visibility of system status
Match user's language
User control/freedom
Consistency
Error prevention
Recognition rather than recall
Efficiency
Aesthetic/minimalist design
Error recovery
Help/documentation
```

## 5-second test

Show page screenshot for 5 seconds.

User should answer:

```text
What is this page?
What can I do here?
What is the most important thing?
```

## Task tests

Run these tasks:

```text
Find chicken prices at Willys.
Check if a deal is real.
Open a meat market trend.
Find the cheapest store for a product.
Open a store from the map.
Compare OTC paracetamol prices.
Compare diesel prices.
Set a price alert.
```

## Success metrics

```text
task completion rate
time to first useful click
search zero-result rate
search-to-product-click rate
deal-card product-click rate
watchlist add rate
page bounce rate
ad accidental-click risk
```

## UI beauty review

Evaluate:

```text
visual consistency
spacing rhythm
typography hierarchy
color harmony
card alignment
icon consistency
image quality
empty states
mobile polish
```
