## BI / Data Analyst Portfolio Case Study

GroceryView is also designed as a **Data Analyst / BI Analyst portfolio project**.

The project turns grocery price data into business-ready reporting outputs:

- chain price index
- basket cost comparison
- product price history
- weekly price movement
- deal detection
- Buy / Wait signal
- data quality monitoring
- automated weekly reporting workflow

### Analyst skills demonstrated

| Skill | Evidence |
|---|---|
| SQL | Reporting views for KPIs, basket costs, price history, and data quality |
| Power BI | Dashboard pages for executive summary, basket comparison, product trends, and data quality |
| Data modelling | Product, chain, store, basket, price observation, and mapping tables |
| KPI design | Chain price index, basket cost, discount percentage, deal score, freshness |
| Data quality | Missing prices, stale records, duplicate mappings, invalid units, outliers |
| Business reporting | Weekly insight report and recommendation template |
| Automation | Power Automate-style scheduled report and alert workflow |
