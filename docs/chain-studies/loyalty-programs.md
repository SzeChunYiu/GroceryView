Last reviewed: 2026-05-24. Scope is GroceryView's Swedish grocery chains where a shopper-facing loyalty or membership program is visible from public chain pages. Use this as source material for the upcoming `/account/loyalty` UI; do not auto-enrol a user or imply a price is available without the membership gate.

## Chains reviewed with no separate loyalty row in the UI

- **Mathem**: no chain-wide grocery loyalty program comparable to the above was confirmed in the reviewed public search results. Keep Mathem out of `/account/loyalty` until a current official membership page is found.
- **Store-specific independent grocers**: many may run local clubs; only add them when an official enrollment URL, cost, benefits and data terms are available.

## UI implementation notes

1. Show `cost_label` separately from `membership_price_numeric`; Coop's 100 kr is a refundable stake, not a recurring fee.
2. Split `benefits` from `data_tradeoff` so the account UI can show a privacy disclosure before outbound enrollment.
3. Add `source_url` and `last_reviewed_at` to each rendered card; enrollment URLs and terms change frequently.
4. Never imply GroceryView can enroll the user. CTA copy should be outbound: “Open enrollment page”.
