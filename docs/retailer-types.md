# Retailer types

These values badge chains on product comparison rows and should describe the retailer's primary merchandising focus.

| retailer_type | Use for |
| --- | --- |
| `supermarket` | Full-line neighborhood grocery stores. |
| `discount` | Discount-led grocery chains with a limited or value-focused assortment. |
| `hypermarket` | Large-format grocery/department stores with broad non-food ranges. |
| `convenience` | Small convenience stores whose grocery range is secondary to immediate needs. |
| `pharmacy` | Pharmacy-led retailers selling grocery-adjacent health and personal care items. |
| `fuel_convenience` | Fuel-station convenience stores. |
| `online` | Online-first grocery retailers or delivery-only grocery catalogs. |
| `ethnic_asian` | Specialty grocers focused on East Asian, Southeast Asian, or pan-Asian assortments. |
| `ethnic_polish_eastern_european` | Specialty grocers focused on Polish, Baltic, Balkan, or Eastern European assortments. |
| `ethnic_middle_eastern` | Specialty grocers focused on Middle Eastern, Turkish, Persian, or Levantine assortments. |
| `ethnic_indian_south_asian` | Specialty grocers focused on Indian, Pakistani, Bangladeshi, Sri Lankan, or South Asian assortments. |
| `ethnic_latin` | Specialty grocers focused on Latin American, Mexican, Caribbean, Central American, or South American assortments. |
| `ethnic_african` | Specialty grocers focused on African regional or pan-African assortments. |
| `health_food` | Health-food, organic, supplement, allergen-free, or wellness-led grocery retailers. |
| `kosher_halal` | Retailers whose primary grocery positioning is kosher, halal, or both. |
