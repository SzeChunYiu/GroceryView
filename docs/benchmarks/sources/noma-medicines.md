# Norwegian Medical Products Agency maximum prices

## What it measures

Norwegian maximum-price and stepped-price medicine reference data for regulated prescription medicines.

## What it does not measure

It is not a normal retail quote and actual prices may be lower than the maximum reference.

## Sample API call

```sh
curl 'https://www.dmp.no/en/public-funding-and-pricing/pricing-of-medicines/maximum-price'
```
