# Statistics Sweden CPI

## What it measures

Swedish national CPI/KPI consumption-group index data for consumer price development.

## What it does not measure

It does not report a GroceryView store price or prove current product availability.

## Sample API call

```sh
curl 'https://api.scb.se/OV0104/v1/doris/en/ssd/START/PR/PR0101'
```
