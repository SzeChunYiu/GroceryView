# EU Agri-food Data Portal

## What it measures

European Commission agricultural market prices for upstream and wholesale context across meat, dairy, eggs, cereals, fruit, and vegetables.

## What it does not measure

It is not a supermarket shelf-price source and must not be labelled as GroceryView retail price evidence.

## Sample API call

```sh
curl 'https://agridata.ec.europa.eu/extensions/API_Documentation/API-Guide.html'
```
