# Statistics Norway CPI table 03013

## What it measures

Norwegian CPI by consumption group and month, including food, pharmaceutical products, petrol, diesel, and fuels/lubricants where present.

## What it does not measure

It does not provide retailer shelf prices or GroceryView station observations.

## Sample API call

```sh
curl 'https://data.ssb.no/api/v0/en/table/03013'
```
