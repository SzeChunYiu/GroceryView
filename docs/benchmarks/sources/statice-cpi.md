# Statistics Iceland CPI / HICP

## What it measures

Icelandic CPI/HICP consumer-index categories published by Statistics Iceland.

## What it does not measure

It does not report live GroceryView product, pharmacy, or fuel prices.

## Sample API call

```sh
curl 'https://px.hagstofa.is/pxen/api/v1/en/Efnahagur/'
```
