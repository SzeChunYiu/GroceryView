# Dairy class taxonomy

Seed: `packages/server/seeds/dairyClasses.sql`.

Covers milk by fat percentage (skim, 0.5%, 1.5%, 3.0%, 3.5%), filmjölk, plain/flavoured yogurt, crème fraîche, sour cream, cottage cheese, cream cheese, butter (salted/unsalted), hard cheese (lagrad, Herrgård, Grevé, Präst), fresh cheese (mozzarella/feta), and eggs by pack count and size.

Variant flags are modelled as dairy variant classes: lactose-free, KRAV organic, and EU-organic. Use variants only when package/source evidence explicitly carries the claim.
