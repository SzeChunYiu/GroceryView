# Preem Sweden source notes

## What this source may contain

- **Station-local consumer evidence:** use only station-specific observations (for example a named station page, receipt, photo, or other station-local record) when showing consumer pump prices.
- **Business list prices:** Preem also publishes business/customer list-price material. Keep this in a separate source bucket from consumer pump prices.

## Guardrails

- Do **not** describe Preem as exposing a chain-wide public consumer pump-price feed.
- Do **not** roll a single station observation up to a national Preem price.
- Label every rendered Preem consumer price with the station or evidence scope that produced it.
- Label business list prices as business/list-price evidence, not shopper pump prices.

## UI label

Recommended data-source label: `Preem · station-local consumer evidence + separate business list prices`.
